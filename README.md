# Лабораторная работа №1  
## Нормализация данных (схема "звезда")

### Описание
Данные из CSV (`mock_data`) были преобразованы из одной таблицы в аналитическую модель типа **звезда (Star Schema)**.

---

### Выполнено
- Развернута PostgreSQL (Docker)
- Загружено 10 CSV (10000 строк)
- Создана таблица `mock_data`
- Выделены:
  - факт: `fact_sales`
  - измерения: `dim_customer`, `dim_seller`, `dim_product`, `dim_store`, `dim_supplier`, `dim_date`
- Реализованы DDL и DML скрипты
- Загружены данные в модель

---

### Результат
Построена модель "звезда", позволяющая выполнять аналитические запросы по продажам.
